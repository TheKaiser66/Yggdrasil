# Bedienung von zuul außerhalb von BlueJ

## Kompilieren einer Datei in der Kommandozeile

```sh
javac DATEINAME.java
```
### Kompilieren aller Java-Dateien:
```sh
javac *.java
```

## Ausführen einer kompilierten Datei in der Kommandozeile

```sh
java KLASSENNAME
```

### Starten des Spiels 
```sh
java Game
```
